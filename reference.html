Reference for v1.1.0

The reference docs tell you about the components included on the latest version of Siimple, and how to use them. For check how to download and install Siimple, read the getting started guide.

TYPOGRAPHY

Siimple uses Open Sans as the default font. The default font-size is 16, and the default font-weight is 300. Siimple comes with bold tag, italic, underline and liks. The default text color is #536273, and the emphasis color is #09a0f6.

Headings

Heading comes with a default margin-bottom of 20px, and each tag has his own line-height. Headings also uses a 300 font-weight, instead of bold.

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Boxes

You can create a box adding the class .box to a div.
Hi, this is a box!
You can write multiple lines in a box, and put links too!. 
<div class="box">Hi, ... </div>

Code

Use the <code> tag to show an inline code, or use the <pre> tag if you want to show one or more lines of code. Both tags uses Open Sans font for the code.

For underline text, you can use the <u> tag! 
For underline text, you can use the
<code>&lt;u&gt;</code> tag!

BUTTONS

Add a .btn class to transform your div, a, etc... into a button. The default color for the button is blue, but you can add a .btn-[color] class for modifying the color, where color can be by default green, orange, purple or red.

Default button
Green button
Orange button
Red button
Purple button
<div class="btn">Default button</div>
<div class="btn btn-green">Green</div>
<div class="btn btn-orange">Orange</div>
<div class="btn btn-red">Red</div>
<div class="btn btn-purple">Purple</div>

You can change the button size adding a .btn-small or .btn-big class to the button.

Small button
Big button
<div class="btn btn-small">Small button</div>
<div class="btn btn-big">Big button</div>

Siimple also comes with outline buttons. You can create an outline button by addind the .btn-outline class to your element.

Outline button
<div class="btn-outline">Outline button</div>

FORMS

Input text

Add a class .form-input to your text-based input fields.

<input type="text" class="form-input">

Additionally, you can add a disabled attribute to your input text field:

<input type="text" class="form-input" disabled>

Input submit

You can add the class .form-input to your submit input fields:

<input type="submit" class="form-input" value="Submit form">

Select

Add the class .form-select to your drop-down list tag:

<select class="form-select">
  <option>Element 1</option>
  <option>Element 2</option>
  <option>Element 3</option>
  <option>Element 4</option>
</select>

Textarea

Simply add the class .form-textarea and modify the attribute rows with the number of rows.

<textarea class="form-textarea"></textarea>

ALERTS

You can create a blue alert adding the .alert class to your div. The default code for the alert is blue.

This is an alert, and this is a link inside the alert.
<div class="alert">This is an alert.</div>

You can change the alert color by adding .alert-[color] class to the alert, where [color] can be green, orange or red.

This is a green alert.
This is an orange alert.
This is a red alert.
<div class="alert alert-green">Green alert.</div>
<div class="alert alert-orange">Orange alert.</div>
<div class="alert alert-red">Red alert.</div>

PANELS

A panel consists in one div with the class .panel, and two div inside, the first with the class .panel-head and the second with the class .panel-body.

Panel
This is the content of the panel
<div class="panel">
  <div class="panel-head">Panel</div>
  <div class="panel-body">Content</div>
</div>

TABLES

Basic table

Simply add .table to the class of your table. You can add a row into a <thead>, and this row will be the head of tour table. Tables always comes with full width property.

# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table"> ... </table>

Hover table

Add .table-hover to the class of your table.

# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table table-hover"> ... </table>

Striped rows table

Add .table-striped to the class of your table.

# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item
Row3 Item Item Item
Row4 Item Item Item 
<table class="table table-striped"> ... </table>

Bordered table

Add .table-border to the class of your table.

# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table table-border"> ... </table>

Additionally you can add the colum border adding .table-border-colum.

# Col1 Col2 Col3
Row1 Item Item Item
Row2 Item Item Item 
<table class="table table-border-column"> ... </table>

GRID

Siimple uses a grid with 10 columns, and a max width of 960px. If you want a full width grid, add the grid-fluid class to the main grid container. Add a col-# class to every column, where # is a number from 1 to 10. For example:

Col 10
Col 1
Col 9
Col 2
Col 8
Col 3
Col 7
Col 4
Col 6
Col 5
Col 5
<div class="grid grid-fluid">
  <div class="row">
    <div class="col-10">Col 10</div>
  </div>
  <div class="row">
    <div class="col-1">Col 1</div>
    <div class="col-9">Col 9</div>
  </div>
</div>
