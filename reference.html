Reference for v1.2.0

The reference docs tell you about the components included on the latest version of Siimple, and how to use them. For check how to download and install Siimple, read the getting started guide.

TYPOGRAPHY

Siimple uses Open Sans as the default font. The default font-size is 16, and the default font-weight is 300. Siimple comes with bold tag, italic, underline, liks and small text. The default text color is #353f4a, and the emphasis color is #09a0f6.

Headings

Heading comes with a default margin-bottom of 10px, and each tag has his own line-height. Headings also uses a 300 font-weight, instead of bold.

Code

Use the <code> tag to show an inline code, or use the <pre> tag if you want to show one or more lines of code. Both tags uses Open Sans font for the code.

For underline text, you can use the <u> tag! 
For underline text, you can use the <code>&lt;u&gt;</code> tag!

BUTTONS

Add a .btn class to transform your div, a or button element into a normal button, or add the .btn-outlie class to transform your element to an outline button. You can change the button size by adding a .btn-small or .btn-big class.

You can change the button color by adding a new class to your button with the next style:

<!-- For change the color to a normal button -->
.btn-color { background-color: [color]; }

<!-- For change the color to an outline button -->
.btn-outline-color { border-color: [color]; color: [color]; }
.btn-outline-color:hover { background-color: [color]; color: #ffffff; }

FORMS

Input text

Add a class .form-input to your text-based input fields.

<input type="text" class="form-input">

Additionally, you can add a disabled attribute to your input text field:

<input type="text" class="form-input" disabled>

Input submit

You can add the class .form-input to your submit input fields:

<input type="submit" class="form-input" value="Submit form">

Select

Add the class .form-select to your drop-down list tag:

<select class="form-select">
  <option>Element 1</option>
  <option>Element 2</option>
  <option>Element 3</option>
  <option>Element 4</option>
</select>

Textarea

Simply add the class .form-textarea and modify the attribute rows with the number of rows.

<textarea class="form-textarea"></textarea>

ALERTS

You can create a blue alert adding the .alert class to your div. The default color for the alert is blue, but you can change it by adding .alert-[color] class to the alert, where [color] can be green, orange or red.

PANELS

A panel consists in one div with the class .panel, and two div inside, the first with the class .panel-head and the second with the class .panel-body.

Panel
This is the content of the panel
<div class="panel">
  <div class="panel-head">Panel</div>
  <div class="panel-body">Content</div>
</div>

TABLES
