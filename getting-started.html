Getting Started

Getting started

First of all, you must download your copy of Siimple. You have four methods:

Download from GitHub (recommended)

You can download a zip or a tar.gz with the lates release of Siimple from here.

Using NPM

You can use NPM for download the latest version of Siimple. For this, open a new console window, navigate to you project home and run the next command:

npm install siimple

This will download Siimple and place it on node-modules/siimple. You can simply enter on this folder and copy the css, which have the compiled version of Siimple, to your project public folder.

Note that you must have NPM installed on your computer for download Siimple using this method.

Using Bower

You can use Bower for download your copy of Siimple. Same as NPM, navigate to your project root folder and run the next command:

bower install siimple

You must have bower installed on your computer for use this option.

Clone the repo

Also, you can download Siimple cloning the repository hosted on GitHub. Simply open a new console window and run the next command: 

git clone https://github.com/siimple/siimple.git

If you want to download a stable version, don't use this method, because this repository is in continuous development. You must have git installed on your computer for do this.

Add to your website

Now, if you have your copy of Siimple downloaded, the next step is adding the next line to the head of every HTML document where you want to use Siimple:

<link rel="stylesheet" type="text/css" href="path/siimple.css">

Additionally, you can add the minimized version of Siimple:

<link rel="stylesheet" type="text/css" href="path/siimple.min.css">

Thats all! Now you can check the reference guide for start using Siimple.

Want to contribute?

You can contribute with Siimple installing it and submitting issues and pull requests :)
