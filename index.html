About Siimple

Siimple is a minimal and responsive CSS framework for flat and clean designs, created initially for my projects. I don't want to compete with Bootstrap, because it is insuperable.

Licensed under the MIT License, it means that you can use, modify and redistribute it without restrictions. If you use Siimple in your project I would see it in action, so please send me an email :)
